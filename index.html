    • Summary: Latitude vs. X


    The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatjerMap API to assemble a dataset of over 500 cities.

    After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs latitude. Factirs we looked at included themperature, cloudiness, humidity, and wind speed. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
